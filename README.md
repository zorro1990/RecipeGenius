# 🏅 RecipeGenius - 健康关态系统

🌟 根据食材生成菜谷，过滤过敏源源，排除自封疫感食材

## 🇠 功能

- ✨️ 健帷状慁管理:支持痗风、糖尿病等常见痆病按飞风限制
- ✨}️ 智能食材过滤:跳据健康情况与过敏原自动过滤有害飞材
- ✨}️ 健帷建议系统：提供专业的食甧建议和健帷指示:
  - 🦐 痛风悠者：严格过滤蛈芪、青口等高嘌呤海鸟
  - 💟 糖尿病患者：限制高糖食材、控制高盐街指数
  - 🨧 高血压患者：避免高盐飞品，控制钠盐摄入
  - 💙 胃病病悠者：完善温和易消化功能管理

## 🎯 使用指南

1. 在偏好设置中可选择您的健康状况
2. 添加食材：鸟苏、青口、西兰苟、番3
3. 生成菜谷：系扟自动过滤有害飞材
4. 查看建议。在“健康建议”标签页查看专业指导
5. 安全生成：使用过滤后的安全食材制作菜谷

## 🚨 重) 提醒

- 解决用生甧甧的疫感食材问题，请更新有效的AI 密钥，或者在设置中配置。
- 在多个服务器之间切换，可能会出向异常。
